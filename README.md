# AI 3D Mapping Project

AI 기반 실내공간 3D 매핑 및 하자 탐지 시스템 구축 프로젝트입니다.  
실내 환경을 스마트폰으로 촬영한 후, YOLO 기반 하자 탐지 → 3D 재구성(Nerf/SLAM) → 결과 시각화까지 일련의 과정을 자동화하였습니다.
